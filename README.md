Custom Noclip Delay

Configurable delay before noclip activates in Geometry Dash. Per-level settings with accuracy and death tracking.

Features
- Set custom level % trigger with decimal support
- Per-level persistent settings
- Accuracy tracker and death counter overlay
- Optional death tint flash with adjustable duration
- Min accuracy % and max deaths limits
- Drag-to-position HUD overlay
- Auto safe mode warning on completion

Install
1. Download `CustomNoclipDelay.geode` from Releases
2. Place in `geode/mods/`
3. Launch Geometry Dash

Usage
Open Pause Menu → click spike icon → set target % and options.  
Drag "Drag Me to Position!" label to move HUD.
